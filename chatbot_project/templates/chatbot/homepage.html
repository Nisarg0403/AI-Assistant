About This Project

CONVERSA is a smart chatbot designed for real-time customer query handling with AI-driven voice interaction and lead identification.

This project leverages the Web Speech API for voice capabilities, a keyword-based FAQ system for dynamic responses, and an analytics dashboard to provide actionable business insights.

Project Goals

Interactive Chatbot

Create a responsive chatbot that supports text and voice input for seamless user interaction.

Dynamic FAQs

Implement a database-driven FAQ system with parent-child relationships for detailed responses.

Analytics Dashboard

Visualize query data with an analytics dashboard, including lead insights to demonstrate business value.

Lead Identification

Detect engaged users using keyword-based rules to identify potential leads.

Technical Highlights

Django & Python

Backend powered by Django framework.

AJAX Integration

Real-time chat updates.

Voice Recognition

Web Speech API for AI-driven voice input.

Lead Detection

Keyword-based rules to identify engaged users.

Development Journey

Step 1: UI Design

Designed a sleek, modern user interface using HTML, CSS, and Anime.js to create an engaging and intuitive experience with smooth animations and responsive layouts.

Step 2: Backend Setup

Developed the backend using Django and Python, establishing a robust foundation for handling chatbot logic, database interactions, and API integrations.

Step 3: Chat Logic Implementation

Built the chat logic by initially exploring spaCy for NLP, then opting for a precise word-matching approach, and integrating AI-driven voice input and lead detection capabilities.

Step 4: Testing and Refinement

Performed extensive testing to refine the user experience and analytics dashboard, incorporating user feedback to optimize performance and functionality.
